# Career Compass AI

## Overview
Career Compass AI is an interactive career guidance platform designed to help students and early professionals discover suitable career paths based on their skills, interests, and goals.

The application allows users to describe their experience and aspirations through a conversational interface and then provides structured insights such as recommended career roles, missing skills, and a personalized learning roadmap.

Instead of filling long forms or browsing multiple websites, users can simply explain their situation and receive tailored suggestions to help them move toward their desired career.

---

## Problem Statement
Many students and beginners struggle when deciding which career path to pursue. Even when they know their interests, they often lack clarity about:

- Which career roles match their current skills
- What skills they are missing
- How to build a structured learning plan
- Which resources to follow to achieve their goals

Most existing platforms provide generic advice that does not adapt to a user's individual background.  
Career Compass AI aims to solve this problem by analyzing a user's input and generating personalized career recommendations, skill-gap insights, and structured learning paths.

---

## Objectives
The main objectives of this project are:

- Help users explore suitable career options
- Identify skills mentioned by the user
- Compare user skills with industry requirements
- Highlight missing skills needed for specific careers
- Generate a structured learning roadmap
- Provide useful resources for skill development

---

## Target Users

Career Compass AI is designed for:

- College students exploring career options
- Beginners learning technical skills
- Individuals planning to switch careers
- Developers planning their skill growth

---

## Core Features

- Chat-based career guidance
- Skill detection from user conversation
- Career recommendation system
- Skill gap analysis
- Personalized learning roadmap generation
- Skill visualization dashboard
- Learning resource recommendations

---

## Minimum Viable Product (MVP)

The first working version of Career Compass AI will include:

- Chat interface
- Skill detection
- Career recommendation
- Learning roadmap generation
- Skill dashboard

Additional features may be added later such as resume analysis, interview preparation, and job insights.
